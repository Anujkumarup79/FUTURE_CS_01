# FUTURE_CS_01 - Vulnerability Assessment

## Description
This project identifies web vulnerabilities using Nmap and OWASP ZAP.

## Tools Used
- Nmap
- OWASP ZAP
- Browser DevTools

## Findings
- Missing Security Headers (Medium)
- Cookie Without Secure Flag (High)
- Information Disclosure (Low)

## Conclusion
The application has security misconfigurations that should be fixed to improve security.
